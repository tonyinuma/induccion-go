# Inducción Go — Practicar con la documentación

A short README to explain that I’m practicing Go by following the official documentation.

## Descripción
This project serves as a practice space where I learn and experiment with Go by studying the official documentation and language examples.

## Cómo practico
- I read the official documentation and package examples.
- I follow the Tour of Go for interactive concepts.
- I implement small programs and tests based on examples from the documentation.
- I review and adapt code from the standard library packages to understand them better.

## Resources
- Documentación oficial: https://go.dev/doc/
- Tour of Go: https://tour.golang.org
- Librería estándar: https://pkg.go.dev/std
